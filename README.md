## QR Code Generator

This is to become a Vue component for rendering QR codes as SVGs.

### Features
- Generate QR codes from strings
- Customize error correction levels (ECL)
- Render QR codes as SVGs for seamless integration into web applications
- SVGs can either be rendered as `path`s or `rect`s (with additional styling)
- The text to QR code matrix conversion will be wrapped as a composable function
